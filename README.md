## Mô tả project:

Đây là project xây dựng một chiếc đồng hồ kim chạy theo thời gian thực, project này chỉ áp dụng những kiến thức cơ bản như: `HTML`, `CSS`, `Javascript`, `Git` và `Github`.

## Các chức năng:

- Kim đồng hồ quay theo thơi gian thực
- Thay đổi giao diện:
  - Dark mode
  - Light mode
